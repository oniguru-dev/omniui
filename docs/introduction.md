## Key Features

- **File-based Routing** — pages are files, routes are automatic
- **Server Components** — `'use server'` directive
- **UnoCSS Engine** — TailwindCSS-compatible utilities
- **Asset Pipeline** — PNG, JPEG, GIF, video optimization
- **SEO Optimization** — sitemap.xml and robots.txt
- **Page Transitions** — canvas-based loading animations
- **TypeScript Strict** — full type safety
- **Vercel-ready** — deploy with one command

## Stack

| Layer | Technology |
|-------|-----------|
| Runtime | Bun |
| Server | Elysia.js |
| UI | Preact |
| CSS | UnoCSS (TailwindCSS engine) |
| Language | TypeScript (strict) |
